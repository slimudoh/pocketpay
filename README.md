# pocketpay

> the pocketpay website

## Install

Install with [npm](https://www.npmjs.com/)

```sh
$ npm i pocketpay --save-dev
```

## Usage

```js
var pocketpay = require('pocketpay');
```

## Running tests

Install dev dependencies:

```sh
$ npm i -d && npm test
```
